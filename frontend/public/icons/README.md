# PWA icons

These are **placeholder** icons for the Gilbert PWA — a centered
white "G" on a `#0a0a0a` canvas. Swap them for branded artwork at
any time; the filenames (`gilbert-180.png`, `gilbert-192.png`,
`gilbert-512.png`, `gilbert-512-maskable.png`) are referenced from
`frontend/vite.config.ts` (web manifest) and `frontend/index.html`
(apple-touch-icon).

## Regenerating

The placeholders were generated by `frontend/scripts/gen_icons.py`
(Pillow):

```bash
# from the repo root
uv run python frontend/scripts/gen_icons.py frontend/public/icons
```

The maskable variant insets the glyph to ~58% of the canvas so OS
masking (Android, iOS) can crop to a circle/squircle without
clipping. Standard icons fill ~72% of the canvas.

## Sizes

| File                          | Size  | Purpose                                     |
| ----------------------------- | ----- | ------------------------------------------- |
| `gilbert-180.png`             | 180²  | iOS apple-touch-icon                        |
| `gilbert-192.png`             | 192²  | Web manifest — small (Android home)         |
| `gilbert-512.png`             | 512²  | Web manifest — large (splash, app drawer)   |
| `gilbert-512-maskable.png`    | 512²  | Web manifest — maskable (purpose=maskable)  |

Note: Vite copies everything under `frontend/public/` verbatim into
`dist/`, so this README ships to production too. That's fine — a
few hundred bytes and harmless. The Python generator deliberately
lives outside `public/` (under `frontend/scripts/`) so it does NOT
ship.
